# Incubyte Gmail Compose Assessment

## Objective

To test Gmail Compose functionality for sending an email with:

* Subject: Incubyte
* Body: QA test for Incubyte

## Deliverables

This repository contains:

1. Traditional Test Cases
2. BDD Test Cases
3. Positive and Negative Test Scenarios

## Files Included

### 1. Incubyte_Test_Cases_Excel.xlsx

Contains traditional/manual test cases covering:

* Gmail Compose functionality
* Recipient validation
* Subject validation
* Email body validation
* Attachments
* Draft functionality
* CC/BCC functionality
* Error handling
* Positive scenarios
* Negative scenarios

### 2. Incubyte_BDD_Test_Cases.xlsx

Contains BDD (Behavior Driven Development) test scenarios written using:

* Given
* When
* Then

covering both positive and negative test cases.

## Application Under Test

Gmail – Compose Email Functionality

## Test Design Techniques Used

* Functional Testing
* Positive Testing
* Negative Testing
* Boundary Testing
* Validation Testing
* Exploratory Testing


Submitted as part of the Incubyte Software Craftsperson Assessment.
